Welcome to TinyBasicBlazor, an experimental Blazor [https://docs.microsoft.com/en-us/aspnet/core/blazor/?view=aspnetcore-3.1] WebAssembly [https://webassembly.org/] Tiny Basic [https://en.wikipedia.org/wiki/Tiny_BASIC] environment by marco's retrobits.
Based on TinyBasic.NET [http://www.thisiscool.com/tinybasicnet.htm], Mohan Embar [http://www.thisiscool.com]'s C# port of Tom Pittman [http://www.ittybittycomputers.com/]'s Tiny Basic [http://www.ittybittycomputers.com/IttyBitty/TinyBasic/index.htm] interpreter C rewrite [http://www.ittybittycomputers.com/IttyBitty/TinyBasic/TinyBasic.c] .

TinyBasic interpreter Copyright 1976 Itty Bitty Computers, used by permission.

Want to learn more about Tiny Basic and included programs? Go to Tom Pittman's Tiny Basic [http://www.ittybittycomputers.com/IttyBitty/TinyBasic/index.htm] page.
